La programación orientada a objetos presenta un paradigma de programación completamente diferente con el que se puede trabajar de una manera más práctica y eficiente. En este capítulo hablaremos un poco sobre cómo se implementan los conceptos de la POO en python.

Clases y objetos ¶

En la programación orientada a objetos (POO) el concepto más importante es el de clase ya que es con esta con la que se debe partir para construir cualquier objeto. Podemos pensar en una clase como un 'plano de construcción' en el que se define qué y cómo debe funcionar cierto tipo de objetos. Una clase contiene todas las funciones y variables necesarias para que un objeto realice las acciones que necesitamos que haga.

Un objeto es una instancia de la clase, o sea, un elemento que existe dentro de nuestro programa y fue construido usando las especificaciones de la clase. Si aún no te queda cien por ciento claro, más adelante daremos ejemplos de esto.

Dentro de una clase tenemos dos elementos muy importantes, lo atributos y los métodos. Hasta este momento nos hemos estado refiriendo a muchos métodos como 'funciones' y es que en el fondo son muy parecidos pero cabe destacar que sólo los métodos están asociados a los objetos. Por otro lado, los atributos son variables que alojan las características específicas de un objeto.

Tanto los atributos como los métodos se 'llaman' usando el nombre del objeto, un punto y el nombre del atributo o el método.

Veamos un ejemplo de cómo crear una clase usando la palabra reservada class.

class NuevaClase:
    def __init__(self, atrib1, atrib2): # Este método se llama constructor y se ejecuta cada que
        self.atrib1 = atrib1            # se crea un objeto de esta clase, aquí definimos nuestros atributos '
        self.atrib2 = atrib2

    def primerMetodo(self):
        print(self.atrib1 + self.atrib2)

Como observaste, el primer 'método' en ser definido es init() también llamado constructor ya que se ejecuta siempre que se crea un nuevo objeto de su clase y define los atributos que contendrá. Los parámetros que recibe son self y el valor de los atributos. Self es una palabra que hace referencia al mismo objeto que se está creando y no tenemos que indicarla al crear un objeto. El valor de los atributos lo proporcionamos al crear el objeto y se guardarán en los atributos que están ahí definidos. Veamos un ejemplo de cómo crear un objeto de esta NuevaClase.

objeto1 = NuevaClase(5,10) # Creamos el objeto1 del tipo NuevaClase con los atributos numéricos 5 y 10

print(objeto1.atrib1) # Imprimimos un atributo
print(objeto1.atrib2) # Imprimimos el otro atributo

objeto1.primerMetodo() # Llamamos al método definido en nuestra clase

Es así cómo se crea una clase y un objeto que contiene los atributos y métodos definidos en esta. Veamos un ejemplo práctico.

class Persona:
    def __init__(self, nombre, edad, sexo):
        self.nombre = nombre
        self.edad = edad
        self.sexo = sexo
    
    def informacion(self):
        return (self.nombre + " tiene " + str(self.edad)+ " años de edad y es del sexo "+ self.sexo)
    
p1 = Persona("Antonio Gutierrez", 25, 'masculino')

p1.informacion()

Herencia ¶

La herencia es otro de los pilares de la POO ya que con ella podemos crear nuevas clases a partir de otras. La idea detrás de esto es muy intuitiva, si creas una nueva clase que hereda de otra ya existente la nueva clase tendrá los atributos y métodos de la clase ya existente más los que se definan en ella. Veamos un ejemplo usando la clase Persona que ya hemos definido.

class Estudiante(Persona):
    def __init__(self, nombre, edad, sexo, carrera): #El constructor recive todos los parámetros iniciales
        Persona.__init__(self,nombre, edad, sexo) #Se mandan los parámetros útiles para la clase 'padre'
        self.carrera = carrera # Se crea un nuevo atributo llamado 'carrera' que sólo existe en esta nueva clase

    def estudios(self): #Este es un nuevo método único de la clase Estudiante
        print(self.nombre + ' estudia la carrera en '+ self.carrera)

La manera de utilizar esta clase es la siguiente:

e1 = Estudiante("Bernardo Jimenez", 21, "masculino", "Ciencias sociales") #Se proporcionan los valores de los atributos

print(e1.informacion()) #Método originario de la clase Persona
e1.estudios() #Método original de la clase Estudiante

Por último vamos a ver cómo modificar un atributo de un objeto ya existente. Simplemente debemos utilizar el nombre del atributo a modificar y asignarle otro valor, de la forma:

e1.nombre = "Miguel Savedra"
